# ElectroVerse - Wishlist & Recommendation System

A shopping app that allows users to add products to a wishlist and suggests products based on their browsing history.

## Features

- **Wishlist Functionality**: Add/remove products from your wishlist and manage them
- **Recommendation System**: Get personalized product recommendations based on your wishlist and browsing history
- **Social Sharing**: Share your wishlist with friends via social media or direct link
- **Product Search**: Find products easily with the search functionality
- **Responsive Design**: Works on desktop and mobile devices

## Technologies Used

- React with Vite
- React Router for navigation
- FontAwesome for icons
- Local Storage for data persistence
- Amazon Affiliate API integration (simulated)
